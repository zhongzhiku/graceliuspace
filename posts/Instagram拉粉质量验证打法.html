用 Website 7-day Click 逻辑，反推 Instagram 拉粉质量

一套绕开 Page Like 1-day click 限制的实战组合打法

在 Meta 广告体系里，很多投手都会遇到一个看似"怪异"、但其实非常系统性的问题：

Instagram 的 Page Like 数据，几乎只能看到 1-day click。

不是你不会设置，也不是账号出问题，而是 Meta 的规则本身就这么设计。

那问题来了：

当拉粉只能看 1-day click，我们该如何判断这些粉到底值不值？

答案是：

👉 不用 Page Like 本身判断，而是用 Website 的 7-day click，反推粉的质量。

一、为什么 Page Like 被锁死在 1-day click？

先把底层逻辑说清楚。

1. Page Like 属于「平台内即时行为」

在 Meta 看来，点赞或关注是一个非常短路径行为：

  • 看到广告
  • 点进去
  • 点 Like

系统判断：这种行为不需要 7 天思考周期

所以：

  • Page Like
  • Engagement
  • IG 互动类目标
  • 👉 默认只给 1-day click（有时 +1-day view）

这不是歧视，是算法定位。

2. 只有 Website 转化，才允许长归因窗口

Website 行为包括：

  • View Content
  • Add to Cart
  • Lead
  • Purchase

这些行为具备：

  • 延迟决策
  • 回访路径
  • 多天思考

因此 Meta 才开放：

  • 7-day click
  • 且往往只在 Website + Maximize Conversions 下可选

3. 结论很明确

广告目标 7-day click
Page Like / Engagement ❌
Lead（Instant Form） ❌
Website + 最大转化率 ✅

所以，与其纠结为什么 Page Like 没有 7-day click，不如承认规则，然后绕开它。

二、核心思路：用 Website 当"质量验证器"

这套打法的核心逻辑只有一句话：

Page Like 只负责把人拉进来，Website 的 7-day click 负责验证这些人是不是"对的"。

换句话说：

  • IG 拉粉是入口
  • Website 转化是体检报告

三、整体结构：三层漏斗打法

第 1 层：Instagram 拉粉（不追 ROI，只追人）

  • 目标： Page Like 或 Engagement
  • 重点：
    • Instagram only
    • 强人设、强价值、强关注理由
  • 数据预期：
    • 接受 1-day click
    • 只看：
      • CPM
      • Like / Engagement 成本
      • 评论是否像"真人在说话"

这一层，不做价值判断。

第 2 层：互动人群沉淀（筛粉）

创建 Custom Audience：

  • 过去 7 / 14 / 30 天
  • IG 视频观看
  • 帖子互动
  • Profile Visit

这一步非常关键。

因为：不是所有关注你的人，都值得进入下一层。

第 3 层：Website 转化验证（拿回 7-day click）

  • Campaign 设置
    • 目标：Website Conversions
    • 优化方式：Maximize conversions
    • Attribution：7-day click
  • 投放人群
    • 第 2 层沉淀的 IG 互动人群
  • 转化事件选择
    • 小账号：View Content / Add to Cart
    • 正常体量：Lead / Purchase

这一层，你终于重新获得：

  • 7-day click
  • 长决策周期
  • 真实意图信号

四、如何反推 IG 粉的质量？

不用复杂模型，只看三件事。

  1. IG → Website CTR
    • 高：粉是"活的"
    • 低：刷到即走型粉
  2. Website 7-day click 转化率
    • 高：粉有决策能力
    • 低：情绪粉 / 娱乐粉
  3. CPA 变化趋势

    如果你发现：

    • IG 拉粉成本不高
    • Website 转化 CPA 反而下降

    👉 说明你拉的是高质量粉，不是水。

五、给老板的解释话术（可直接用）

"IG Page Like 本身只能看 1-day click，但我们用 Website 的 7-day click 来验证这些粉丝后续是否真的产生行为。如果 7 天内仍然能转化，说明拉粉质量是成立的，而不是表面好看。"

这句话，能解决 80% 的质疑。

六、进阶用法：创意反推

你可以：

  • 用不同创意拉 IG 粉
  • 成本接近
  • 丢进同一个 Website 7-day click 漏斗

最后你会看到一个很现实的结果：

最便宜的粉，往往是最贵的。

七、什么时候这套打法最有价值？

  • 品牌冷启动
  • IG 粉涨得快，但不转化
  • Page Like 被 1-day click 锁死
  • 老板怀疑"粉是不是水的"

总结一句话

Page Like 是门面数据，Website 7-day click 是价值验证。

前者负责规模，后者负责真相。
